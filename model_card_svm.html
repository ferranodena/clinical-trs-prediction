Model Card for SVM RBF per Predicció de Resistència al Tractament (TRS)

Model Details

Overview

Model predictiu basat en Support Vector Machine (SVM) amb kernel RBF dissenyat per identificar pacients amb esquizofrènia que presenten resistència al tractament antipsicòtic estàndard. El model prioritza la sensibilitat per actuar com una eina de cribratge en la decisió clínica.

Model Parameters

Model Architecture

Support Vector Machine (SVM) amb kernel RBF

Input Format

Hiperparàmetres Clau: Kernel = RBF, C = 0.1, Gamma = auto, Class Weight = balanced.

Considerations

Use Cases

  • RECOMANACIÓ: Utilitzar com a eina de cribratge (screening) per identificar pacients amb alt risc.
  • ADVERTÈNCIA: Els resultats positius han de ser sempre validats per un especialista degut a la taxa de falsos positius.
  • ADVERTÈNCIA: No utilitzar per a la presa de decisions automàtica sense supervisió humana.

Limitations

  • COMPROMÍS PRECISIÓ-RECALL: El model sacrifica precisió global per garantir la detecció de la majoria de pacients TRS.
  • SENSIBILITAT A L'ESCALAT: Requereix un preprocessament (StandardScaler) rigorós per funcionar correctament.
  • GENERALITZACIÓ: El rendiment pot variar segons les característiques demogràfiques de la població aplicat.

Datasets

TRS Dataset (Clinical + Genetic)

Entrenat amb dades clíniques i genètiques de pacients amb esquizofrènia. Inclou dades demogràfiques, clíniques, biomarcadors i marcadors genètics. Nota: Aquestes dades es consideren sensibles (informació mèdica).

Quantitative Analysis

Accuracy - 0.56

Description: Precisió global al conjunt de prova (Test Set).

Recall (TRS) - 0.63

Description: Capacitat del model per detectar correctament els pacients resistents (Sensibilitat).

Precision (TRS) - 0.38

Description: Proporció de prediccions "TRS" que són realment correctes.

F1-Score (TRS) - 0.48

Description: Mitjana harmònica entre precisió i recall per a la classe minoritària.

AUC - ROC - 0.62

Description: Capacitat de discriminació real entre les dues classes per sobre de l'atzar.
